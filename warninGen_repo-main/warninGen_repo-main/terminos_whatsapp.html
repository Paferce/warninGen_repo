IMPORTANTE: Este es un canal solo de notificaciones. No es necesario ni posible responder a las alertas por WhatsApp.

5. Privacidad

El servicio es GRATUITO. Solo pueden aplicar costos de datos de tu operador móvil.

Utilizamos WhatsApp Business API de Meta/Twilio. Requiere conexión a internet.

9. Responsabilidad

No garantizamos entrega instantánea (depende de cobertura, internet, etc.). Siempre mantén activos otros medios de información (radio, TV, web oficial).
